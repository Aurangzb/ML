# ML - Machine Learning Projects Portfolio

A comprehensive collection of machine learning projects, algorithms, and agentic AI implementations showcasing practical applications of AI and ML concepts.

## 📁 Repository Structure

### 📊 Projects
This directory contains end-to-end machine learning projects:

- **AiLogoGenerator**: Generates creative logos using GAN technology. The model learns from existing logo designs to generate novel, unique logos.
  - `ModelGan.py` - GAN model architecture
  - `DataLoader.py` - Data loading and preprocessing
  - `Train.py` - Training pipeline
  - `Compiler.py` - Model compilation utilities

- **DiamondClassification**: Predicts and classifies diamonds based on various features using machine learning techniques. Complete analysis and model training pipeline included.
  - `diamond_classification_problem.ipynb` - Complete Jupyter notebook with data analysis, model training, and evaluation

### 🤖 AgenticAI
Intelligent agent implementations leveraging agentic patterns and LLMs:

- **PersonalCareerAgent**:An intelligent agent powered by LLMs that provides personalized career guidance, advice, and recommendations for professional development.
  - `personal_career_agent.py` - Python implementation of the career agent
  - `personal_career_agent.ipynb` - Jupyter notebook with examples and documentation

### 📚 ML Algorithms
Educational implementation of the K-Nearest Neighbors algorithm with training examples and utilities.

- **KNN (K-Nearest Neighbors)**: Implementation and training examples
  - `knn_algorithm.py` - KNN algorithm implementation
  - `train.py` - Training and evaluation script

### 💬 QandA Bot
A conversational Q&A system combining LangChain with FAISS vector search for efficient document retrieval and question answering.

- **pythonProject**: LangChain-based Q&A application
  - `langchain_helper.py` - LangChain integration utilities
  - `main.py` - Application entry point
  - `requirements.txt` - Python dependencies (LangChain, FAISS, etc.)
  - `faiss_index/` - Vector index storage
